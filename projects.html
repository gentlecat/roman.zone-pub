Projects

Sometimes I program a computer in my spare time.

At this point, I've got a bunch of side projects that might be worth sharing. Some of them are unfinished or unpolished (or most of them, to be more accurate). But I made this list hoping that some of these might be useful to someone, or just to satisfy the curiosity of those who want to know what else I'm doing.

Steam History
2013
This was built to keep track of usage changes for each app available on the Steam platform. Usage history is not part of the official statistics, but current active user counts are available through their web API. This app records usage history for each available app and provides a nice interface to view it.
  • Go
  • SQLite
Publisher
2016
Blog generator that was built for publishing my own articles, but can be used for a similar purpose by anyone else. Allows to create custom templates, supports Markdown with a bunch of fancy features.
  • Go
Cazador de Puertos
2017
Web app for tracking participants of the hill climbing challenge organized by a cycling club in Barcelona. Based on the Strava API, written in Go.
  • Go
Counter
2011
Just a regular tally counter for Android. This was the first app that I built when I started learning to program. It got surprisingly popular, currently installed on over 90k active devices. 🤔
  • Android
  • Java
Steam Locomotive
2012
Steam Locomotive is a PHP wrapper for the Steam Web API with useful tools such as Steam ID converters, validators, etc. Originally used as a part of a never released project, then released as a separate library.
  • PHP
Cache Emulator
2013
Demo for a fully associative CPU cache with LRU replacement policy. aka how to make boring university assignment interesting.
  • C++
  • Qt
